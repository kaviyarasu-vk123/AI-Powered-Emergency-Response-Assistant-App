# 🚑 ResQ - AI Emergency Response Assistant

## 💡 Problem
In emergency situations, delayed response can cost lives. Many people don’t know what to do before medical help arrives.

## 🚀 Solution
ResQ is an AI-powered mobile application that provides:
- Instant SOS alert
- Emergency detection using AI
- Step-by-step first aid guidance
- Real-time location sharing

## ⚙️ Tech Stack
- Frontend: Flutter
- Backend: FastAPI (Python)
- AI: Whisper + BERT (NLP)
- Database: Firebase + SQLite
- APIs: Mapbox, Twilio

## 📱 Features
- One-tap SOS button
- Cardiac emergency detection
- First-aid instructions
- Live location sharing

## 🔮 Future Scope
- Wearable integration
- Real-time ambulance tracking
- Voice-based emergency detection
